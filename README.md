# ProyectosDocumentos

Plugin para [FacturaScripts](https://facturascripts.com) que permite añadir archivos a proyectos.

> Fork actualizado de [DocumentosProyectos](https://facturascripts.com/plugins/documentosproyectos) original de [oguimera_10015](https://facturascripts.com/perfiles/10015), adaptado para compatibilidad con FacturaScripts 2025.

## Requisitos

- FacturaScripts 2025 o superior
- Plugin [Proyectos](https://facturascripts.com/plugins/proyectos)

## Instalación

1. Descarga el archivo `.zip` desde [facturascripts.com](https://facturascripts.com/plugins/proyectosdocumentos)
2. Ve a **Plugins** en tu instalación de FacturaScripts
3. Sube el archivo `.zip` e instala

## Funcionalidad

Añade una pestaña de archivos adjuntos dentro de la ficha de cada proyecto, permitiendo:

- Subir archivos relacionados con el proyecto
- Añadir una descripción a cada archivo
- Los archivos quedan también disponibles en la biblioteca general de FacturaScripts
- Desvincular o eliminar archivos del proyecto
